# myLifeTracker
A mobile life-tracking app that visualises daily progress through a growing tree. Log your health, habits, and routine and  watch your tree grow.

-Screens include

-Today — dashboard with pulse score, bonsai tree, and notices

-Physical — log sleep, water, and movement daily

-Routine — manage tasks, meetings, and deadlines

-Habits — build daily habits with streaks and XP

-Onboarding — four-slide intro for first-time users

-Tech Stack
React Native + Expo go
AsyncStorage (offline, on-device data)
React Native SVG
